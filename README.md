# CSI-MMCOE

Website url: https://csi-mmcoe.org/

# Usage

Install dependencies (Tailwind)

```
npm install
```

Run the Tailwind CLI to compile the **input.css** during development. The output file is **css/main.css**

```
npm run build
```

To build once run...

```
npm run watch
```

To run website

```
python manage.py runserver
```
